Example 1: Retreiving JSON using AJAX

This demonstration uses the AJAX technology to import
a JSON file and create a story from some of its
properties. There is some added CSS animation and
transition for flavor. Click the button to import
the JSON object and let the JavaScript function
create the story.

Example 2: Local Storage API, Storing and Retrieving
and DOM Manipulation

This example will demonstrate the storing and retrieving of
of an object as well as DOM manipulation of the "Thank You"
message at the very bottom of the example. Once you are done
adding items to your cart and clear your cart, you may then
reload that data stored in the local storage by clicking RELOAD.
This example uses local storage rather than session storage so the
information is not lost when you close this tab.
